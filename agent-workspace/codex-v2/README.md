# Codex Multi-Agent V2

This directory holds durable operator artifacts for the additive Codex v2
worker mesh. It does not replace Fleet Pi's Pi-backed browser chat runtime.

- `plans/` contains explicit plan artifacts generated by `pnpm codex-v2:plan`.
- `runs/` contains machine-readable run state.
- `reports/` contains human-readable run summaries.
- `traces/` is reserved for trace exports and per-run debugging notes.

Mutating workers require an explicit `pnpm codex-v2:execute -- --run-id <id>`
gate. Live Codex MCP dispatch is additionally guarded by `--use-codex` so local
validation and dry runs remain deterministic.
