## Contributing

The repository is open for contributions. If you want to report a bug, propose a feature, or submit a fix, open a GitHub issue or pull request at [https://github.com/Qredence/fleet-pi](https://github.com/Qredence/fleet-pi). Please read `CONTRIBUTING.md` before submitting a pull request, and follow the code quality guidelines in `AGENTS.md`.

Security issues should not be reported as public GitHub issues. See `SECURITY.md` for the private reporting process.
